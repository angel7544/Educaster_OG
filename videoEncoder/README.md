# EduCaster (Local Encoder)

A production-ready Python application that converts video files into multi-quality HLS streams (VOD/Live) and uploads them directly to Cloudflare R2.

## Features

- **Adaptive HLS**: Converts video into 1080p, 720p, 480p, 360p, 240p.
- **Direct R2 Upload**: Uploads segments (.ts) and playlists (.m3u8) to Cloudflare R2 *while* encoding.
- **Live Streaming Mode**: optimized HLS flags for low latency and playlist management.
- **Hardware Acceleration**: NVENC support.
- **Parallel Processing**: Encodes multiple resolutions simultaneously.
- **Java Integration**: Can be launched directly from the Educater Java Desktop App.

## Prerequisites

1. **Python 3.7+**
   Install dependencies:
   ```bash
   pip install boto3 watchdog requests
   ```

2. **FFmpeg** 
   Must be in your system `PATH`.

## Usage

### 1. From Educater App (Recommended)
1. Open the **Cloudflare R2** tab in Educater.
2. Click **Upload**.
3. Select your video file.
4. Check **"Optimize for Streaming (HLS) via EduCaster"**.
5. Click **Start Upload**.

The app will launch the encoder in the background, transcode the video to HLS, and upload it to your R2 bucket.

### 2. GUI Mode (Standalone)
Run the script to open the GUI:
```bash
python convert.py
```
- Select Input/Output.
- Toggle "Live Streaming Mode" if needed.
- Click Start.

### 3. CLI Mode
```bash
# VOD Mode (Standard)
python convert.py input.mp4 output_dir/ --parallel --single_folder

# Live Mode (Short segments, delete old segments, master playlist first)
python convert.py rtmp://localhost/live/stream output_dir/ --live
```

## Configuration

The R2 credentials are loaded from `r2_config.json`.
When launched from the Educater App, this file is automatically generated with your current credentials.

If using standalone, create `r2_config.json`:
```json
{
    "r2_endpoint_url": "https://<ACCOUNT_ID>.r2.cloudflarestorage.com",
    "r2_access_key_id": "<ACCESS_KEY>",
    "r2_secret_access_key": "<SECRET_KEY>",
    "r2_bucket_name": "<BUCKET_NAME>",
    "r2_public_domain": "https://pub-<ID>.r2.dev"
}
```
